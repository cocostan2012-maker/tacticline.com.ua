# TacticLine — незалежний каталог

Це версія сайту без посилань на OLX. Сайт працює як власний резервний каталог товарів з повними описами, пошуком, категоріями, кошиком-заявкою та замовленням через Telegram.

## Як оновити сайт на GitHub Pages

1. Розпакуй ZIP.
2. Відкрий свій репозиторій на GitHub.
3. Натисни **Add file → Upload files**.
4. Перетягни файли з розпакованої папки:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `products.js`
   - `README.md`
5. Натисни **Commit changes**.
6. Через 1–3 хвилини GitHub Pages оновить сайт.

## Як додати товар

Відкрий `products.js`. Кожен товар — це окремий блок `{ ... }`.

Головні поля:

- `title` — назва товару
- `category` — категорія
- `price` — ціна в гривнях або `null`, якщо треба "ціну уточнюйте"
- `description` — короткий опис на картці
- `fullDescription` — повний опис у вікні "Детальніше"
- `specs` — параметри
- `package` — комплектація
- `tags` — короткі мітки

## Контакт для замовлення

За замовчуванням використовується Telegram: `@NordicKostya`.

Щоб змінити Telegram, відкрий `app.js` і зміни рядок:

```js
const TELEGRAM_USERNAME = "NordicKostya";
```
