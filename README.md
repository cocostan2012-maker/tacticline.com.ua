# TacticLine — безкоштовний сайт-каталог

Це статичний сайт-магазин без серверу: працює на HTML, CSS і JavaScript. Його можна безкоштовно розмістити на GitHub Pages, Netlify або Vercel.

## Що всередині

- `index.html` — головна сторінка.
- `styles.css` — дизайн.
- `app.js` — пошук, фільтри, кошик і Telegram-заявка.
- `products.js` — список товарів.

## Як змінити товари

Відкрий `products.js` і редагуй блоки товарів:

```js
{
  id: "unikalnyy-id",
  title: "Назва товару",
  category: "Категорія",
  price: 130,
  stock: "наявність уточнюйте",
  tags: ["тег 1", "тег 2"],
  description: "Короткий опис товару",
  olxUrl: "https://www.olx.ua/uk/list/user/Q6uhw/",
  icon: "★"
}
```

Якщо ціна невідома — постав `price: null`.

## Як безкоштовно опублікувати через GitHub Pages

1. Створи акаунт на GitHub, якщо його немає.
2. Створи новий репозиторій, наприклад `tacticline-store`.
3. Завантаж усі файли з цієї папки в репозиторій.
4. Перейди в `Settings` → `Pages`.
5. У `Build and deployment` вибери `Deploy from a branch`.
6. Branch: `main`, folder: `/root`.
7. Після збереження GitHub дасть посилання на сайт.

## Як оновлювати сайт

Змінив `products.js` → завантажив оновлений файл у GitHub → сайт сам оновиться.

## Примітка

Сайт не приймає оплату і не створює реальні замовлення на сервері. Кошик формує текст заявки і відкриває Telegram продавця.
